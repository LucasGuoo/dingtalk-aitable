# 对接案例沉淀

本目录用于存放钉钉AI表格对接的历史案例，包括：

- 完整的对接场景描述
- 调用的接口清单和示例
- 遇到的问题和解决方案
- 优化建议和最佳实践

## 案例列表

| 案例编号 | 场景描述 | 对接日期 | 状态 |
|---------|---------|---------|------|
| TEMPLATE | 模板文件 | - | 模板 |

## 案例编号规范

格式：`CASE-YYYYMMDD-NNN`

- CASE: 固定前缀
- YYYYMMDD: 对接日期
- NNN: 当日序号(001-999)

## 如何添加新案例

1. 复制 `TEMPLATE.md` 文件
2. 按编号规范重命名文件
3. 填写案例内容
4. 更新本README中的案例列表
